Getting Started with CSS Grid Layout

Chinomnso 8 min read

CSS Grid Layout is a two-dimensional layout system designed for the web. It lets you lay out items in rows and columns, and has many features that make building complex layouts straightforward.

Why Use CSS Grid?

Before CSS Grid, we used floats, positioning, and inline-block to create layouts. These methods were never designed for complex layouts, which made them difficult to use and limited in functionality.

CSS Grid was specifically designed for layouts, making it much more powerful and intuitive. Here are some benefits:

  • Two-dimensional control over both rows and columns
  • Ability to place items precisely
  • Control of spacing between items
  • Overlap of items when needed
  • Named grid areas for easier placement

Basic Grid Terminology

Before diving into code examples, let's understand some basic terminology:

  • Grid Container: The element with display: grid applied to it.
  • Grid Item: The direct children of the grid container.
  • Grid Line: The dividing lines that make up the grid structure.
  • Grid Cell: The space between four grid lines.
  • Grid Track: The space between two adjacent grid lines (rows or columns).
  • Grid Area: The space surrounded by four grid lines.

Creating Your First Grid

Let's start with a simple example:

.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  grid-template-rows: 100px 100px;
  gap: 10px;
}

This creates a grid with three columns of 200px each and two rows of 100px each, with a 10px gap between all grid items.

Using Flexible Units

Instead of fixed pixel values, you can use flexible units like fr (fraction):

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto;
  gap: 20px;
}

This creates a grid with three columns where the middle column takes up twice as much space as the others.

Placing Items on the Grid

You can control where items are placed on the grid:

.item {
  grid-column: 1 / 3; /* Start at line 1, end at line 3 */
  grid-row: 2 / 3; /* Start at line 2, end at line 3 */
}

This places the item to span from column line 1 to 3 and from row line 2 to 3.

Conclusion

CSS Grid is a powerful layout tool that makes complex layouts much easier to create. This article just scratches the surface of what's possible with Grid. In future articles, we'll explore more advanced features like named grid areas, auto-placement, and responsive grid layouts.

Start experimenting with Grid in your projects, and you'll quickly see how it can simplify your layout code while enabling designs that were difficult or impossible before.
